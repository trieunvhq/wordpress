== Description ==

Float Left Right Advertising can scroll up/down banner when user scroll page up/down. Support HTML, image, flash. Multi options. 

Want to see Demo? visit http://blog.casanova.vn

Features:

* Can scroll up/down banner when user scroll page up/down 
* Can set width/height of banner left or right
* Can set show Ads on mobile devices or not
* Enable html code for Ads

== Installation ==

1. Download the plugin.
2. Upload it to the plugins folder of your blog.
3. Goto the Settings --> Float Left Right Advertising to config your Ads.
